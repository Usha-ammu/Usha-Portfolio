FUSIONQ-A-HYBRID-INTELLIGENCE-FRAMEWORK-FOR-HIGH-QUALITY-AUTOMATIC-QUESTION-GENERATION

The FUSIONQ framework is a Hybrid Intelligence system designed to generate high-quality, automatic questions (AQG) for educational and assessment purposes. It integrates the robust capabilities of Large Language Models (LLMs) for text generation with symbolic, rule-based reasoning to guarantee grammatical accuracy and topic relevance, overcoming the common pitfalls of purely neural approaches. This powerful fusion ensures the creation of diverse and contextually appropriate questions, significantly enhancing the rigor of automated content creation. By automating the question generation process while enforcing quality controls, FUSIONQ dramatically reduces instructor workload and promotes scalable, effective learning experiences. This makes it an essential tool for modern e-learning platforms demanding high assessment integrity.

YUMICE: Interactive Ice Cream E-commerce Showcase (Figma Design)

This project is a responsive web design prototype created in Figma for a fictional premium ice cream shop, "YUMICE." The site features a clean, modern layout with a prominent hero section for featured flavors like Strawberry Vanilla and Gelato Four Scoop. A key interactive element allows users to cycle through different ice cream flavors by clicking on the thumbnails, dynamically updating the main image, description, and price. The design aims to maximize visual appeal and user engagement through bright imagery and clear calls-to-action like "Order Now" and "See Menu," demonstrating strong UI/UX design skills.
